## How to use
1. Add it in root `build.gradle`
```
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
2. Add it in module `build.gradle`
```
dependencies {
    implementation "com.github.entan05:eLibraries:$VERSION"
}
```
